puregram

puregram is a powerful Node.js package that allows you to easily interact with Telegram API ðŸš€

Examples

Features
  • 100% Telegram API coverage
  • Works with JavaScript and TypeScript
  • Typings

Installation

Node.js 12.0.0 or newer is required

Yarn

 
yarn add puregram

NPM

 
npm i puregram

Example usage

 
import { Telegram, MessageContext } from 'puregram';

const telegram: Telegram = new Telegram({
  token: process.env.TOKEN
});

telegram.updates.on('message', (context: MessageContext) => {
  context.send('Hi!');
});

telegram.updates.startPolling().catch(console.error);
